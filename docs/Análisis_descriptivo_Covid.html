Se evidencia que el campo de fecha de inicio de s√≠ntomas se est√° utilizando para marcar los casos de personas asintomaticas pero se pienza utilizar este campo para realizar los conteos de infecciones porlo cual es necesario hacer un tratamiento con los registros con el valor 'Asintom√°tico'

Inicialmente convertimos los campos de fechas a tipo de dato fecha con el parametro de forzado lo cual nos generar√° unos valores NaT para los valores que no sean covnvertidos a fecha y posteriormente dichos valores se reemplazan con la fecha de reporte web

In [15]:
df['Fecha de muerte'] = pd.to_datetime(df['Fecha de muerte'])
df['Fecha diagnostico'] = pd.to_datetime(df['Fecha diagnostico'])
df['FIS'] = pd.to_datetime(df['FIS'], errors="coerce") #Este campo tiene un registro con un valor que no corresponde
df['fecha reporte web'] = pd.to_datetime(df['fecha reporte web'])
df['Fecha de notificaci√≥n'] = pd.to_datetime(df['Fecha de notificaci√≥n'])
df['FIS'].fillna(df['fecha reporte web'], inplace=True)

Es necesario crear los campos de los contadores segun las columnas de fecha que tengan valores para poder realizar los conteos posteriormente.

In [16]:
# Se crean los contadores
df["Casos"] = 1
df.loc[(df['Fecha de muerte'].notnull() == True), 'Muertos'] = 1 
df['Muertos'].fillna(0, inplace = True)
df.loc[(df['Fecha recuperado'].notnull() == True), 'Recuperados'] = 1
df['Recuperados'].fillna(0, inplace = True)
df.loc[(~df['Fecha de muerte'].notnull() == True) & (~df['Fecha recuperado'].notnull() == True), 'Activos'] = 1 
df['Activos'].fillna(0, inplace = True)

df.loc[(~df['Fecha de muerte'].notnull() == False) & (~df['Fecha recuperado'].notnull() == False), 'ERR'] = 1 
df['ERR'].fillna(0, inplace = True)
